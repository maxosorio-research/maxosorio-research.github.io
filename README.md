# Maximiliano Osorio Web

Sitio personal estático de Maximiliano Osorio Requena, listo para publicarse en GitHub Pages.

## Estructura

- `index.html`: página principal.
- `assets/css/styles.css`: estilos del sitio.
- `assets/js/main.js`: navegación móvil y animaciones de aparición.
- `assets/img/`: foto de perfil y favicon.
- `assets/docs/cv-maximiliano-osorio.pdf`: CV descargable.
- `robots.txt`, `sitemap.xml`, `site.webmanifest`: archivos básicos para publicación.
- `404.html`: página para rutas inexistentes en GitHub Pages.

## Publicación en GitHub Pages

1. Subir el contenido de esta carpeta a un repositorio.
2. Activar GitHub Pages desde `Settings > Pages`.
3. Para publicarlo como sitio principal, usar el repositorio `maxosorio-research.github.io`.
4. Si el nombre final del repositorio es distinto, actualizar las URLs canónicas en `index.html`, `robots.txt` y `sitemap.xml`.
